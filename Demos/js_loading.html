Démonstration JavaScript: Chargement des scripts

Contenu avant chargement des scripts

Cette page contient deux scripts JavaScript et du contenu affiché avant et après le chargement et l'exécution de ces scripts:


<!DOCTYPE html>
<html>
<head>
    <title>JavaScript démo</title>
    ...
</head>
<body>
    <main>
        <h1>Démonstration JavaScript: Chargement des scripts</h1>
        <section>
            <h2>Contenu avant chargement des scripts</h2>
            ...
        </section>

        <script src="https://javascript.info/article/script-async-defer/long.js?speed=1"></script>
        <script src="https://javascript.info/article/script-async-defer/small.js"></script>

        <section>
            <h2>Contenu après chargement des scripts</h2>
            ...
        </section>
    </main>
</body>
</html>

Le premier script simule un chargement long (3 secondes) et le second un chargement rapide (0.5 seconde).

Contenu après chargement des scripts

Cette partie de la page s'affiche après le chargement et l'exécution des scripts.

Pour solutionner ce problème, deux possibilités:

  1. Ajouter l'attribut defer sur les balises <script>: le contenu s'affiche immédiatement et les scripts s'exécutent ensuite, dans l'ordre.
  2. Ajouter l'attribut async sur les balises <script>: le contenu s'affiche immédiatement mais le script court s'exécute avant le script long.
